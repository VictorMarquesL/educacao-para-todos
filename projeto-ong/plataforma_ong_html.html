# Educação para Todos 🏫 Projeto acadêmico desenvolvido como parte da disciplina de **Fundamentos e Estruturação de Sistemas Web**, com o objetivo de aplicar conceitos de **HTML5 semântico** e **estruturação de pastas** em uma página institucional de uma ONG fictícia voltada à educação. --- ## 📚 Sobre o Projeto A **Educação para Todos** é uma ONG fictícia que tem como missão democratizar o acesso à educação para jovens e crianças em situação de vulnerabilidade. O projeto consiste em um site institucional simples, estruturado em **HTML5** e **CSS3**, composto por três páginas principais: ### 🔹 Páginas - **index.html** – Página inicial com informações sobre a ONG (missão, visão, valores e contato). - **projetos.html** – Apresenta os projetos sociais e programas oferecidos pela ONG. - **cadastro.html** – Formulário de cadastro de voluntários e doadores, com validação de campos. --- ## 🗂️ Estrutura de Pastas ``` /projeto-ong │ ├── index.html ├── projetos.html ├── cadastro.html └── /assets ├── /css │ └── style.css └── /img ``` --- ## 🎨 Tecnologias Utilizadas - **HTML5** – Estrutura e semântica das páginas - **CSS3** – Estilização e layout responsivo --- ## ⚙️ Como Executar o Projeto 1. Baixe ou clone este repositório: ```bash git clone https://github.com/seuusuario/educacao-para-todos.git ``` 2. Acesse a pasta do projeto: ```bash cd educacao-para-todos ``` 3.
